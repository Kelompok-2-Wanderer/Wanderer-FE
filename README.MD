## 🧰 Installation

Clone this project
```bash
  git clone https://github.com/Kelompok-2-Wanderer/Wanderer-FE.git
  cd Wanderer-FE
```

#### ⌛ Install
```bash
  npm install
```

#### 🚀 Usage
```bash
  npm run dev
```
